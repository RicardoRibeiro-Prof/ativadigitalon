Agora adicione estas quatro variáveis em Vercel → Settings → Environment Variables:

  • ADMIN_PASSWORD
  • SESSION_SECRET
  • GITHUB_TOKEN

Depois faça um novo Redeploy. A senha e o token não aparecem no código do site.
